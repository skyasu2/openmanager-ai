# Reports History (Legacy Snapshot)

과거 Claude Code 중심 워크플로우에서 생성된 레거시 문서 스냅샷 보관 경로입니다.

## 현재 역할

- 기본 원칙: `read-only` 레거시 보관소
- 신규 계획서/완료본 저장 경로로 사용하지 않음
- 현재 표준 경로:
  - 진행 중: `reports/planning/*.md`
  - 완료본: `reports/planning/archive/`

## 정리 원칙

- 이 경로의 문서가 현재 운영에 필요하면 활성 경로로 승격 후 정리합니다.
- 더 이상 필요하지 않은 레거시 문서는 삭제 가능하며, 삭제 전에 필요한 내용만 활성 문서로 이관합니다.
- 최신 변경 기록은 루트 `CHANGELOG.md`를 우선 참조합니다.

## 디렉토리 예시

```text
reports/history/
├── changelog/
└── completed-plans/
```
