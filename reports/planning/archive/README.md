## next-tasks-plan 이력

`next-tasks-plan.md`는 현재 진행 중인 작업 계획의 SSOT. 완료 시 아래 규칙으로 아카이브:

| 날짜 | 파일 | 주요 내용 |
|------|------|----------|
| 2026-04-07 | [next-tasks-plan-2026-04-07.md](next-tasks-plan-2026-04-07.md) | TypeScript 6.0.2, Knip v6, Storybook hygiene, node suite 최적화, v8.11.0 릴리스 |
| 2026-04-14 | (현재 next-tasks-plan.md) | Advisor Agent 품질·지연 개선 (latency 임계값, 프롬프트, quality-retry) |

## 아카이브 규칙

- `next-tasks-plan.md`가 **Status: Completed Canonical**로 변경될 때 아카이브
- 파일명: `next-tasks-plan-YYYY-MM-DD.md` (완료 날짜 기준)
- 이 README 테이블에 항목 추가

## 기타 문서 분류

| 패턴 | 설명 |
|------|------|
| `YYYY-MM-DD-work-history.md` | 날짜별 작업 이력 요약 |
| `*-plan.md` | 특정 기능/영역 계획서 (완료 후 보관) |
| `REFACTORING-PLAN-*.md` | 리팩토링 계획서 |
