# Commit Message Patterns

## Examples

- `feat(stitch): add incremental prompt template`
- `fix(dashboard): prevent stale top-alert ordering`
- `refactor(ai-engine): split incident parser logic`
- `docs(workflow): add stitch improvement guide`
- `chore(mcp): adjust health check output`

## Guidance

- Keep subject line under ~72 chars when possible.
- Use imperative tone.
- Scope should match changed area.
