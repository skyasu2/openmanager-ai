# PR Template

## Title pattern

`<type>(<scope>): <summary>`

Examples:

- `fix(stitch): align registry validation with current workflow`
- `docs(mcp): add codex-safe deployment guide`

## Body minimum

- What changed
- Why changed
- Risk/impact
- Verification commands executed

## Verification section example

```text
Validation
- npm run test:quick
- npm run type-check
- npm run lint
```
